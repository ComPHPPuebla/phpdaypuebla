Comunidades

Comunidad PHP Puebla

El objetivo de la Comunidad PHP Puebla es el de reunir desarrolladores PHP que quieran aprender y compartir experiencias con otros desarrolladores.

El plan es reunirnos una vez por mes para hablar de algún tema sobre PHP o programación Web que la misma comunidad elija.

Comunidad PHP México

Somos una comunidad de Programadores de PHP, que tienen como objetivo el tener un espacio para generar nuevos proyectos de software usando la plataforma de PHP.




Hackers & Founders Puebla

Somos un grupo de programadores que nos reunimos para discutir temas de tecnología y compartir buenas prácticas de programación.
